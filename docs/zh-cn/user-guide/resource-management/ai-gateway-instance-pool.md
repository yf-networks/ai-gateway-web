# AI网关实例池

## 概述

AI网关实例池用于维护AI网关转发引擎的实例列表。每个实例通过机器名、IP/域名和端口进行标识，请求由这些实例接收并转发。

在左侧菜单选择「资源管理」→「AI网关实例池」，进入AI网关实例池管理页面。

![AI网关实例池页面](../../../images/AI-gateway-resource-instance.png)

## 实例池配置

实例池页面展示当前已配置的实例列表。每行实例包含以下字段：

- **机器名**：实例标识名称，必填。
- **IP/域名**：实例访问地址，支持 IPv4、IPv6 或域名（FQDN），必填。
- **端口**：实例访问端口，默认端口为 80，范围为 1～65535，必填。

![AI网关实例池详情页面](../../../images/AI-gateway-resource-instance-detail.png)

### 编辑实例池

1. 点击页面右上角「编辑」按钮，进入编辑模式。
2. 在表格中直接修改已有实例的机器名、IP/域名和端口。
3. 点击表格下方的「+ 添加」按钮，可以新增实例。
4. 点击每行末尾的「删除」按钮，可以移除对应实例。实例池至少保留一个实例，最后一个实例不可删除。
5. 修改完成后点击「提交」保存；点击「取消」放弃当前修改并刷新页面。

### 校验规则

- 机器名、IP/域名、端口均为必填项。
- IP/域名需为合法的 IPv4、IPv6 地址或域名。
- 端口必须在 1～65535 范围内。
- 同一实例池内不允许存在重复的「IP/域名 + 端口」组合。

### 注意事项

- 实例池为全局单一配置，页面直接展示当前生效的实例池内容。
- 编辑模式下至少保留一个实例，否则无法提交。
